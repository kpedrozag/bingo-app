# bingo-app

This is a repository to store the source code of the apps to play BINGO.
- `cartones-app` for https://cartones-bingo-kp.netlify.app
- `sorteador-app` for https://sorteador-bingo-kp.netlify.app

## Tech requirements

- NodeJS v16.15.0.

These apps were built using [Svelte framework](https://svelte.dev).

## How to run?

1. Go to the folder of each app: `cd cartones-app` | `cd sorteador-app`
2. Run `npm i`
3. Run `npm run dev`
4. Go to http://localhost:8080.
5. Have fun

## How to contribute?

1. Create [an issue](https://github.com/kpedrozag/bingo-app/issues).
2. Clone the repository, create a new branch, push your changes and publish the branch.
3. Create a Pull Request.
4. Ask an admin to review your PR and approve.
5. Merging the PR will trigger the CD.
